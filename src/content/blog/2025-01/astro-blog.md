# Astro 정적 블로그

Astro는 **zero JS by default** 원칙으로 빠른 웹사이트를 만들 수 있습니다.

## 주요 기능

1. **마크다운 네이티브** - Content Collections로 타입 안전한 콘텐츠 관리
2. **태그 & 카테고리** - frontmatter로 분류
3. **월별 아카이브** - 폴더 구조로 체계적 관리

## 마크다운 예시

| 기능 | 설명 |
|------|------|
| 태그 | 세부 주제 분류 |
| 카테고리 | 상위 주제 분류 |

**굵은 글씨**와 *기울임*도 지원합니다.
